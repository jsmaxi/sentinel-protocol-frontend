## Sentinel Decentralized Application

## Quick Start:

```
cd sentinel-dapp
npm install
npm run dev
```

Open: https://localhost:3000/

### Scaffold:

```
npx create-next-app@latest sentinel-dapp
cd sentinel-dapp
npm run dev
```

Frontend Technologies:

- NextJs

- TailwindCss

- @stellar/freighter-api

- @stellar/stellar-sdk
